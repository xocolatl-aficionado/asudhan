Hi!

I’m a Cloud Engineer focused on keeping microservices reliably deployed, and antifragile. With certifications in AWS, production experience in Azure and personal work on GCP, I’ve found myself curious about the strengths and weaknesses of various clouds, and how a growing company tackles the trade-off between owning their own cloud vs renting another. Particularly, I’m fascinated by how ubiquitous open source tools like Kubernetes and Terraform, coupled with paradigms like Continuous Integration and standards like Yet Another Markup Language, have transformed the way applications are built, tested and deployed, dramatically reducing production delivery times.
